Την εβδομάδα που μας πέρασε έγινε μελέτη πανω στην Java και στο προγραμμα για test spy framework  Mockito αλλά υπήρχαν
κάποιες μικρές επιπλοκές οπότε δεν μπόρεσα να υλοποιήσω κάποιο παραδειγμα.Για αυτό τον λόγο θα πώ κάποια
περιληπτικά πράγματα για το πως λειτουργεί!
Το Mockito όπως προανέφερα βοηθάει στο να κάνεις test με την τεχνική double-test και την βοήθεια τον mock-object στις κλάσεις που έχεις δημιουργήσει.
Το θέμα είναι να ξέρεις σε ποιό αντικείμενο θές να κάνεις test και ποιό είναι το αντικείμενο που σε βοηθάει να κάνεις test.
Τα αντικείμενα που σε βοηθάνε να κάνεις test ονομάζονται stub objects και τα αντικείμενα στα οποία κάνεις το
test ονομάζονται mock objects.Γενικά οποιοδήποτε αντικείμενο που δεν κάνεις mocking επάνω του και βοηθάει
να τρέξεις το test ονομάζεται stub object!
Στο αντικείμενο που κάνεις assert είναι και το mock object το οποίο τεστάρεις!
Στην περίπτωση που έχεις κάνει mock σε πολλά αντικείμενα mock object θα θεωρηθει το αντικείμενο στο οποίο έχεις
κάνει assert.
Οι κατηγορίες όλων τον εργαλειών που χρησιμοποιεί το Mockito είναι οι εξής(περιλαμβάνονται και τα προηγούμενα):
1)Dummy Object
2)Test Stub
3)Test Spy
4)Mock Object
5)Fake Object
